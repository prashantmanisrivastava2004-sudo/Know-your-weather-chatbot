# 🌤️ Weather Forecast Chatbot

A simple and interactive **Weather Forecast Chatbot** built using **Python, Flask, and Dialogflow**. The chatbot allows users to enter a city name and receive the current weather conditions instantly.

## ✨ Features

* 🌍 Get real-time weather information for any city
* 🌡️ Displays temperature and weather conditions
* 💬 Conversational interface powered by Dialogflow
* ⚡ Fast and lightweight Flask backend
* 📱 Simple and user-friendly design

## 🛠️ Tech Stack


* **Backend:** Python, Flask
* **Chatbot Platform:** Dialogflow
* **API:** OpenWeatherMap API




## 🚀 How It Works

1. User opens the chatbot interface.
2. User enters a city name.
3. Dialogflow processes the request.
4. Flask fetches weather data from the OpenWeatherMap API.
5. The chatbot responds with the current weather details.
